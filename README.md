# Truth Social 日本語化 + 手動翻訳（Chrome拡張）

Truth Social のUIラベルは**ローカル辞書のみ**で日本語化し、投稿本文・コメント本文は**翻訳ボタン押下時のみ**翻訳して**原文 + 翻訳の二段表示**にする Manifest V3 対応の Chrome 拡張です。

## 0.6.2 の修正
- **プロフィール自己紹介の翻訳ボタン機能を完全削除**
- 投稿ボタン判定ロジックを再レビューし、日本語投稿・動画主体投稿での誤表示をさらに抑制
- UI辞書翻訳は 0.5 系の短文・ラベル中心の置換ロジックを維持
- 投稿下部ボタン、翻訳文だけの折りたたみアニメーションは維持
- MutationObserver 起因の重複ボタンを出しにくくするよう既存ボタンの除去ロジックを追加
